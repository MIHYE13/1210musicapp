# 🎹 화음 분석 & 피아노 연주 - 빠른 시작 가이드

## 5분 만에 시작하기! 🚀

### 1단계: 설치 (1분)

```bash
# 다운로드 후
unzip elementary-music-helper-v5.0.zip
cd elementary-music-helper

# 패키지 설치
pip install -r requirements.txt

# YouTube 기능 사용시 (선택)
pip install yt-dlp
```

### 2단계: 실행 (30초)

```bash
streamlit run src/app.py
```

브라우저 자동 실행: `http://localhost:8501`

### 3단계: 화음 분석 페이지로! (30초)

```
왼쪽 사이드바
→ "🎹 화음 분석 피아노" 클릭
```

## 🎵 첫 화음 분석 (3분)

### 방법 1: MIDI 파일 사용

```
1. "🎵 MIDI 반주 분석" 탭 클릭

2. MIDI 파일 업로드
   - 반주 파일 (accompaniment.mid)
   - 또는 샘플 MIDI

3. 자동 분석 시작!
   ✅ 다장조 변환
   ✅ 화음 추출
   ✅ 피아노 건반 표시

4. 결과 확인
   마디 1: C (도미솔)
   마디 2: G (솔시레)
   마디 3: Am (라도미)

5. 피아노 클릭!
   → 🔊 소리 재생
```

### 방법 2: YouTube URL 사용

```
1. "📺 YouTube 음원" 탭 클릭

2. URL 입력
   https://youtube.com/watch?v=...

3. "오디오 다운로드 및 분석 시작" 클릭

4. 자동 처리
   → 다운로드 (30초)
   → 멜로디 추출
   → 화음 분석

5. 피아노 건반 확인!
```

## 🎹 인터랙티브 피아노 사용법

### 기본 사용

```
1. 화음 코드 확인
   예: "C 코드 (도미솔)"

2. 피아노 건반 보기
   [C] [E] [G] ← 금색으로 표시

3. 클릭하여 소리 듣기
   마우스로 도 → 미 → 솔 클릭
   → 화음 소리 확인!

4. 연습
   여러 번 반복
   천천히 익히기
```

### 마디별 연습

```
1. 슬라이더로 마디 선택
   "연습할 마디 선택: 1"

2. 해당 마디 화음 확인
   마디 1: C (도미솔)

3. 피아노 건반 표시
   도, 미, 솔이 금색

4. 순서대로 클릭 연습
   도 → 미 → 솔
   함께 → C 화음!

5. 다음 마디로
   슬라이더: 2
   마디 2 연습
```

## 💡 Pro Tips

### Tip 1: MIDI 파일 구하기

**무료 MIDI 사이트:**
- https://musescore.com (Creative Commons)
- https://freemidi.org
- https://bitmidi.com

**다운로드:**
```
1. 사이트 방문
2. "학교종" 검색
3. MIDI 다운로드
4. 앱에서 업로드
```

### Tip 2: YouTube 활용

**최적의 영상:**
- 피아노 반주 영상
- MR (반주 음원)
- 느린 템포 영상

**예시 검색어:**
```
- "학교종 피아노 반주"
- "학교종 MR"
- "학교종 반주 느리게"
```

### Tip 3: 화음 학습 순서

**초급 (추천):**
```
1. C 코드: 도미솔
2. G 코드: 솔시레
3. F 코드: 파라도
4. Am 코드: 라도미
```

**연습:**
```
1. 각 코드를 피아노에서 찾기
2. 천천히 클릭하며 소리 확인
3. 순서대로 연결하기
4. 반복 연습
```

### Tip 4: 다운로드 활용

```
1. 화음 악보 다운로드 (MIDI)
   → MuseScore에서 열기

2. 화음 악보 다운로드 (MusicXML)
   → 다른 앱에서 사용

3. 화음 진행표 캡처
   → 학생들에게 공유
```

## 🔧 문제 해결

### "MIDI 파일을 분석할 수 없어요"

**원인:**
- 멜로디만 있는 파일
- 화음 정보 없음

**해결:**
```
1. 반주가 있는 MIDI 사용
2. 또는 메인 페이지에서 반주 추가
3. 그 후 화음 분석
```

### "피아노 소리가 안 나요"

**원인:**
- 브라우저 미지원
- Web Audio API 차단

**해결:**
```
1. Chrome 또는 Edge 사용
2. HTTPS 연결 확인
3. 브라우저 권한 허용
```

### "YouTube 다운로드 실패"

**원인:**
- yt-dlp 미설치
- 네트워크 오류

**해결:**
```
1. pip install yt-dlp
2. 수동 다운로드 가이드 참조
3. MP3로 변환 후 업로드
```
