# MP3 파일 악보 생성 지원

## ✅ MP3 파일 지원 가능합니다!

MP3 파일로 악보 생성을 지원합니다. 두 가지 방법이 있습니다:

## 지원 방법

### 1. basic-pitch 사용 (권장, 더 정확함)
- **상태**: 구현 완료
- **설치**: `pip install basic-pitch`
- **주의**: Python 3.14에서는 설치가 어려울 수 있음
- **해결책**: Python 3.10-3.12 사용 권장

### 2. librosa 사용 (대안, 현재 활성화됨)
- **상태**: 구현 완료 ✅
- **설치**: `pip install librosa` (이미 설치됨)
- **장점**: Python 3.14에서도 작동
- **특징**: 피치 추출을 통한 멜로디 생성

## 현재 구현 상태

### ✅ 작동하는 기능
1. **MP3 파일 업로드** → 악보 생성
2. **WAV 파일 업로드** → 악보 생성
3. **자동 폴백**: basic-pitch 실패 시 librosa 자동 사용

### 처리 순서
```
1. basic-pitch 시도 (더 정확함)
   ↓ 실패 시
2. librosa 사용 (대안)
   ↓ 성공
3. 악보 생성 완료
```

## 사용 방법

### API 엔드포인트
```
POST /api/audio/process
Content-Type: multipart/form-data
Body: file (MP3, WAV 파일)
```

### 지원 파일 형식
- ✅ MP3
- ✅ WAV
- ✅ MIDI (직접 변환)
